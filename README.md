## ⚙️ How it works

This web app generates random passwords using a combination of:

- Uppercase letters (A–Z)
- Lowercase letters (a–z)
- Numbers (0–9)
- Symbols (e.g., !@#$%^&*)

You can customize password length and choose which character types to include.
